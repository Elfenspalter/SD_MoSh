# SD_MoSh
My Sales &amp; Dungeons Stuff for the Mothership RPG

## Dice Roller
Simple Diceroller, which rolls 1W1, 1-4W10, 1W20 and 1W100

## Character Generator
Generates random Mothership Characters. Class and gender can be selected, rest is random.
